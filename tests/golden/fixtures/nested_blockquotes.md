# Nested Blockquotes

Blockquotes containing headings, lists, code, and other blockquotes as children.

## Blockquote with Heading Child

> ## Overview
>
> This blockquote contains a heading and a paragraph.

## Blockquote with List Child

> Key points:
>
> - First bullet point
> - Second bullet point
> - Third bullet point

## Blockquote with Ordered List Child

> Steps to follow:
>
> 1. Prepare the environment
> 2. Run the command
> 3. Verify the output

## Blockquote with Code Child

> Example usage:
>
> ```python
> result = compute(x, y)
> print(result)
> ```

## Nested Blockquote (Two Levels)

> Outer quote level one.
>
> > Inner quote level two with **bold** text.

## Mixed Children in Blockquote

> A starting paragraph with *italic* text.
>
> - List item inside quote
> - Another list item
>
> Closing paragraph of the outer quote.
