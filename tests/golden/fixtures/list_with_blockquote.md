# Lists with Blockquotes

Lists containing blockquotes as nested children.

## Bulleted List with Quote

- First item
- Second item with a remark:

  > This is a quoted note inside the list.

- Third item

## Numbered List with Quote

1. Step one
2. Step two with a warning:

   > Important: do not skip this step.

3. Step three

## Task List with Quote

- [x] Completed task
- [ ] Pending task with context:

  > Waiting on approval from the team.

- [ ] Another pending task
