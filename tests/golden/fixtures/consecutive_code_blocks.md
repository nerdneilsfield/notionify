# Consecutive Code Blocks

Multiple code blocks appearing back-to-back without prose between them.

```python
def hello():
    return "world"
```

```javascript
function greet() {
    return "hello";
}
```

```bash
echo "done"
```

## Mixed Languages Back-to-Back

```json
{"key": "value"}
```

```yaml
key: value
```

x = 42
