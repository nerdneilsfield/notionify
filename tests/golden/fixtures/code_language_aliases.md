# Code Language Aliases

Short aliases should be normalised to their canonical Notion language names.

## Python short alias

```py
def greet(name: str) -> str:
    return f"Hello, {name}!"
```

## JavaScript short alias

```js
const double = x => x * 2;
```

## TypeScript short alias

```ts
function greet(name: string): string {
    return `Hello, ${name}!`;
}
```

## Shell short alias

```sh
echo "hello world"
```

## Ruby short alias

```rb
puts "Hello, Ruby!"
```

## Rust short alias

```rs
fn main() {
    println!("Hello, Rust!");
}
```
